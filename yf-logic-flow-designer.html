How to Build:
1️⃣ Drag blocks onto the canvas
2️⃣ Fill in the details on each block
3️⃣ Connect blocks by dragging from ● to ●
4️⃣ Decision blocks have Yes / No paths
5️⃣ Hit ▶️ Run It! to simulate!
Block Types:
🟢 Start/End — Every flow needs these
⚡ Action — A step that does something
❓ Decision — If-then branching
🔄 Loop — Repeat things
📦 Variable — Remember values
📋 List — Group of items
➕ Operator — Math & logic
🎤 Input — Ask the user
💬 Output — Show messages
Tips:
• Click a connection line to delete it
• Select a block and press Delete to remove
• Use 💾 Save to keep your design
• Variables track in the right panel
